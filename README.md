# agentic-ai-study-notes
A personal, ever-growing collection of study notes and code, centered on agentic AI, large language models, and Python. It currently covers autonomous agents, retrieval-augmented generation (RAG), tool use, planning, and memory, with implementations primarily in Python. Beyond these core topics, the repository is designed to absorb new areas of learning — from additional programming languages to adjacent fields like quantitative finance, ML engineering, or system design — making it a long-term, expandable research notebook.
